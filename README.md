# Stock-performance-and-Risk-Analysis
Multi-Stock Performance &amp; Risk Analytics A Python-based tool to analyze and visualize stock performance for Tech Giants (AAPL, MSFT, TSLA, etc.). It calculates daily returns, evaluates annualized volatility (risk), and generates Matplotlib bar charts for data-driven investment insights.
Key Features:
Real-time Data: Leverages the yfinance library to fetch historical closing prices directly from the live market.
Daily Returns Calculation: Performs mathematical analysis to track the daily percentage change for each individual stock.
Risk Assessment: Identifies high-risk stocks by calculating Annualized Volatility (Standard Deviation) across the portfolio.
Data Visualization: Utilizes Matplotlib to generate clear, insightful bar charts for quick performance comparison.
Exportable Results: Provides the ability to save analysis results in CSV format for easy reporting and offline use.
